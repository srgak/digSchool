# DigitalSchool

Проект работает на ангуляре версии 15.2.6. Данный проект находится в разработке.

## Быстрый старт

1. Склонировать репозиторий
2. Сделать npm i
3. Запустить тестовый сервер (npm run server)
4. Запустить клиент (npm run start)
